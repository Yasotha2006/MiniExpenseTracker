# MiniExpenseTracker

A simple and beginner-friendly Expense Tracker Web App built using HTML, CSS, and JavaScript.

It helps users track their daily expenses easily.

**🚀 Features**

➕ Add new expenses

📋 View expense list

💸 Automatically calculate total

🎨 Clean and simple user interface

**🛠️ Technologies Used**

HTML

CSS

JavaScript

**📁 Project Structure**

expense-tracker/

│── index.html

│── style.css

│── script.js

**⚙️ How It Works**

User enters expense name and amount

Clicks Add Expense

Data is stored in an array

Expense list is updated dynamically

Total amount is calculated and displayed

**🧠 Concepts Used**

DOM Manipulation

Event Handling

Arrays & Objects

Loops (for)

Functions
